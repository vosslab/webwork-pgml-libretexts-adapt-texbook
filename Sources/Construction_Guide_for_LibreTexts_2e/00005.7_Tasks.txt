5.7: Tasks

Tasks can be created using the Task bar. The timeline displays the task schedule either as a calendar or a Gannt chart.

Tasks can be set up and added singly (Add Task) or in a batch mode. Each task has a status, a creation date, a start and an end date.

As appropriate, team members can be assigned to the task, status and dates changed, etc. Messages appropriate to the task can be
communicated to people working on the task.

The timeline can be displayed as a Gannt chart or a calender.
