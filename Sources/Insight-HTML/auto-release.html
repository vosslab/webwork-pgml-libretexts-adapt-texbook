Auto-Release Tool for Assignments

Source: https://commons.libretexts.org/insight/auto-release
Last updated at 03/21/2025 12:10 PM by Delmar Larsen

Description: ADAPT's new auto-release system is designed to facilitate "releasing" of assignments to students.

ADAPT's new auto-release system is designed to facilitate "releasing" of assignments to students. Released courses are viewable to students enrolled in the class. This is different from "opening" an assignment, which means that students can start submitting to questions in that assignment. The operating theme of the Auto-release tool is that the instructor will rely on ADAPT to release the assignment to students a fixed time before the assignment opens. 

The Auto-release tool is designed to reduce students being overwhelmed if they see all the assignments of the class at the start of the class and it simplifies the instructors effort by removing the responsibility to manually release assignments. This can be daunting if a lot of assignments exist for the course.

The release status of assignments are indicated on the top level view of the course (see below) via the radio buttons under the "Released Status" column or via the color hue of the assignment. An instructor may release an assignment either manually (instantly) or via the new auto-release system at a set date and time. The "Released Status" of an assignment may be either: 

  • unreleased (students cannot see assignment) - no radio button switched and the assignment has a red hue,
  • delayed (students will see assign at a later date/time) - only "Auto" is switch and the assignment has a orange hue, or
  • released (students can see assignment) - the "Manual" button is switched and the assignment has a green hue. The manual button takes precedence over the "Auto" button so it doesn't matter if the assignment were set for Auto-release or not.

The Control Panel

The Auto-release has a broader scope than just releasing assignments to students a fixed time before the assignment opens. The tool allows for ADAPT to automatically release properties after the close of the assignment including student scores, question solutions, and/or class and individual statistics. These can be manually switched via the "main view" toggle right below the course title. This switches the course view to "Control Panel" view.

The Control Panel is structured similarly to the Main View with explicitly release status and release dates for "Scores", "Solutions" and "Statistics". The assignments are color coded based on the Release Status of the assignment from the main view NOT based on the status of the "Scores", "Solutions" and "Statistics". Those are indicated by the radio buttons switches. For example, in the picture below, the Score, Solutions, and Statistics for " Lecture 1 - Conservation of Energy" are set for 1/8/25 at 9:00PM. However, the Scores, Solutions, and Statistics for the " Click Me! Clicker test Import" assignment will not be released at any time via the Auto-release.

Auto-release Settings

The settings for autorelease can be found in the "Auto-Release" option under the Course Properties.

The auto-release settings is relatively simple and involve selecting the delay and reference date/time ("Condition') for releasing. For assignments, the reference time is set at the "available on" which is the opening time. The example below is set for 5 days before the opening time.

The Scores, Solutions, and Statistics can have variable reference time. If the assignment has a late policy, then there are two deadlines and you can pick either one for the reference date/time.

The "Global Update" provides the applicable assignment that the parameters selected above apply to. They can be all assignments or ADAPT allows instructors to apply different auto-release parameters to different assignments.
