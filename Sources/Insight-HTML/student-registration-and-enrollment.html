Description: Student Registration and Enrollment in ADAPT

How do students register and enroll into their ADAPT courses?

LMS-enabled courses (No Access Codes Required)

If students are trying to enroll in a LMS-enabled course, then the students are automatically registered via their LMS. The LTI protocol allows for ADAPT to pull the roster and create ADAPT accounts that need to be created and enroll students into the associated ADAPT course automatically. Student can only access their ADAPT assignments via their campus LMS (as Modules or Quizzes). They cannot access their assignments directly via the ADAPT website.

Non-LMS-enabled courses (Required Access Codes)

If students are trying to enroll in a LMS-enabled course, then the students are automatically registered for ADAPT and enrolled into their course via their LMS. Students can only access their ADAPT assignments via their campus LMS. They cannot access their assignments directly via ADAPT.

Student-specific Access Codes

  • When students are given student-specific access codes each student is emailed a specific access code that works for that student only (tied to the email address of the access code). If the invitation were set with student names (optional but recommended), then the students have ADAPT accounts created for them automatically. However, if the invitations were created with only emails, then the students need to create an ADAPT account manually (see below) before enrolling with the access code. 

Course-specific Access Codes

  • When students are given course-specific access codes each student is given the same access code that works for any student to enroll in the class. Students will need to create an ADAPT account manually (see below) before enrolling with this access code.

Manually Registering for an account

For a student to manually register for an account (for non-LMS courses only), go to ADAPT website and click on Log in/Register in upper right corner. After the student registers, the student can enroll in the course using the access code (student-specific or course-specific, which ever if applicable). To enroll in the course, sign into the ADAPT website, click on Enroll in Course and the enters the access code online.
