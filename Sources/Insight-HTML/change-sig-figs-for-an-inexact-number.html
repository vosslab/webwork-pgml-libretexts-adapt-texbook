Set Significant Figures for an Inexact number

Last updated at 07/07/2024 03:16 AM by Delmar Larsen

Description: Change Sig Figs for an Inexact number

If you already have the InexactValue created and what to change the sig figs on the fly, a good command is to use the sigFigs method:

$value = InexactValue('2.336');  # has 4 sig figs 

$value->sigFigs(3);  # now it has 3 sig figs and displays 2.34 when printed

CAREFUL! The internal value is still 2.336. It will only display 2.34. But if you multiply it by something else, it will use the entire 2.336 in the calculation.
