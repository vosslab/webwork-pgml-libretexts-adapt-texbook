Description: Coupling ADAPT Courses to Canvas via LTI only

ADAPT courses can be linked to LMS courses via either LTI only or, in the case of Canvas, via a combination of LTI and API. The video below shows the steps for linking ADAPT to Canvas via LTI only. This requires manually construction of quizzes in Canvas and then manually linking each quiz to the corresponding assignment in the ADAPT course. 

The following instructions show how to manually link your course to Canvas:

  1. Within ADAPT, instructors should navigate to a particular course that they would like to use within Canvas.  
  2. Under Course Properties for that course,  choose "Yes" for LMS to enable the LTI grade passback feature: https://youtu.be/watch?v=CjvAzBTRM9o
  3. You can think link up your assignments from Canvas: https://youtu.be/mwptJHUjY6w
