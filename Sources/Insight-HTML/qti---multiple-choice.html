QTI - Multiple Choice

Source: https://commons.libretexts.org/insight/qti---multiple-choice
Last updated at 09/08/2025 11:54 AM by Delmar Larsen

Description: QTI Multiple Choice is a standardized question format that supports creating and exchanging multiple-choice assessment items across different learning platforms.

QTI Multiple Choice is a standardized question type in which learners select a single correct answer from a set of options. It is widely used for assessing knowledge and comprehension.

 

Write a question prompt and then create a selection of answers, choosing the correct answer from the list. Students will receive a shuffled ordering of the selection. Optionally provide feedback at the individual question level or general feedback for a correct response, an incorrect response, or any response.
