ADAPT learning trees are a graph-based approach to adaptive student learning. The tree is a collection of nodes that point to each other (a directed acyclic graph). Nodes can refer to a question, video, notes, or remarks about the contents of the tree. Trees comprise one root node, with branches of varying width and depth. Students first attempt a question associated with the root node. If the student answers the question incorrectly, they are directed to review one branch of the learning tree, to bolster their understanding. Students may attempt the root node question after completing a branch of the learning tree.
