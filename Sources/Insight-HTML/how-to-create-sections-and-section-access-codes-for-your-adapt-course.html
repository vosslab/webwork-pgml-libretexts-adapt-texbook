How to Create Sections and Section Access Codes for your ADAPT Course

Source: https://commons.libretexts.org/insight/how-to-create-sections-and-section-access-codes-for-your-adapt-course
Last updated at 08/07/2025 12:52 PM by Jake Turner

Description: This guide provides a step-by-step process to set up course sections and generate access codes.

1. From "My Courses" (https://adapt.libretexts.org/instructors/courses ) find the correct course and click on the "gear" icon for "Course Properties"

 

 

2. In "Course Properties", click "Sections" in the sidebar

 

 

3. If you don't have an existing section, or need to create another one, click "Add Section"

 

4. Enter a name for the section and a Course Reference Number (CRN). This is a number that identifies a specific section of a course being offered. It does not have to follow any specific format or sequence.

 

 

5. Click "Submit"

 

 

6. Click the "Copy" icon next to the Access Code for the relevant section. This is the code that students will need to enter to join that section of your course in ADAPT
