How do I start writing an assignment in The Forge?

Source: https://commons.libretexts.org/insight/how-do-i-start-writing-an-assignment-in-aw
Last updated at 03/29/2025 03:28 PM by Delmar Larsen

Description: How do I start writing an assignment in The Forge?

  1. Once you are logged in, click on the My Classes (Student) button .
  2. Click on the name of the class for which you wish to write an assignment.
  3. Click on the Write button for the assignment you wish to begin writing.
  4. Click on the Start button to create the document.
