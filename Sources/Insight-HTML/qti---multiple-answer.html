QTI - Multiple Answer

Description: QTI Multiple Answer is a standardized question format that lets learners select more than one correct option from a set of choices, supporting flexible scoring and interoperability across platforms.

QTI Multiple Answer is a standardized question type that allows learners to select more than one correct option from a list of choices. Unlike traditional multiple-choice questions, multiple-answer items can have several correct responses, and scoring can be configured to give partial credit, require all correct selections, or apply other grading rules. This format is widely used for assessing complex understanding and critical thinking, and it supports interoperability across learning platforms that adhere to the QTI (Question and Test Interoperability) standard.
