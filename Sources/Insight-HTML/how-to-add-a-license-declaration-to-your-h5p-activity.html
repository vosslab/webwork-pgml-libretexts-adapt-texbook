How to add a license declaration to your H5P activity

Source: https://commons.libretexts.org/insight/how-to-add-a-license-declaration-to-your-h5p-activity
Last updated at 12/11/2025 10:45 PM by Jake Turner

Description: We encourage all H5P content creators on Studio to set the appropriate OER license declaration so others know how they can reuse your content.

Once signed in to Studio, find your Author Menu in the top-right corner of the screen and select My H5P:

 

 

 

From My H5P find the activity to want to add a license to and select the green Edit button:
 


In the H5P Editor tab, look for a blue Metadata tag, usually next to the Title. The exact location may vary based on the activity type:

 

 

In the Metadata dialog, select a License and (if necessary) a License Version. Then select Save to close the dialog:

 

 

Select Save once again at the top of the page:

 

 

The License and License Attribution details should now be displayed on your activity's overview!
