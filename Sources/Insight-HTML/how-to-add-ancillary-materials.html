Last updated at 01/30/2024 02:04 AM by Delmar Larsen

Description: Ancillary Materials.

How to proceed with adding ancillaries to a LibreTexts OER?

 

1: If the ancillaries are files (e.g., powerpoint presentations), then add them directly to the  conductor page for the project. This will be shown on the Commons page for the project (the catalog) and eventually on the project page itself (soon). You can select if you want the ancillaries to be public - available to all, open to only verified instructors, or private to only project members.

 

2: if the ancillaries are assessment based, they can be stored as files (see above) or they can be converted to questions in ADAPT and/or studio (H5P). If the latter, the exact workflow for adding assessments into ADAPT requires a discussion with the LibreTexts team as several auto-graded technologies. In many cases, the ADAPT harvesting team is available for harvesting questions into ADAPT and converting them into autograded assessments.
