QTI - Select Choice

Source: https://commons.libretexts.org/insight/qti---select-choice
Last updated at 09/08/2025 11:38 AM by Delmar Larsen

Description: QTI Select Choice is a standardized question format where learners choose the correct response from a dropdown list embedded within a sentence or problem, enabling inline answer selection.

Using brackets, place the correct answer in the location that you want the select choice to appear. Then, add the choices below.

Example. [Star Wars] took place in a [galaxy] far, far away.

Note that bracketed words should only appear once. If you need to use the same correct answer multiple times, you can use a dummy identifier such as [Star Wars-1] where the "1" should be increased each time you need to use the same correct answer. Then, below, you can update the correct answer manually.

Each student will receive a randomized ordering of the choices.
