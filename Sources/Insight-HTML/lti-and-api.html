Coupling ADAPT to Canvas via LTI and API

Source: https://commons.libretexts.org/insight/lti-and-api
Last updated at 02/06/2025 08:09 PM by Delmar Larsen

Description: Coupling existing ADAPT courses to existing Canvas courses via the combined LTI and API protocols.

Granting Access

First, you'll need to grant ADAPT access to your Canvas instance. If you have not done so already, in a Canvas enabled course, click on Grant Access:

Then, click Authorize

You'll only have to do this once per school. Here's a video of the process of syncing ADAPT course to Canvas via the LTI+API approach: https://youtu.be/zQgFlXPyHgM

For technical reasons, a course should either be all API+LTI (automatically synced) or LTI only (manually synced). Finally, changing the name of assignments will not break anything. But, the "ADAPT" title will be a thing of the past: with the API, whatever name you provide within ADAPT will be sent off to campus so that the two names are in sync.
 

Linking a New Course

Then in future semesters, you'll just need to copy the Canvas course, copy the ADAPT course and re-link: This video shows the process: https://youtu.be/rBytc6ZEgls

 

FAQ

-         Do I have to re-install the ADAPT LTI after turn on the API key?

 

You don't. Please just follow the instructions provided and leave the LTI key alone: it needs to be active as well.

 

 

-         Do I need to turn off the LTI 1.3 key?

No. Please keep it active.
