There are plenty of guides for writing mathematics assessment questions in WeBWorK. This guide takes a different approach and is written for people in the sciences and other descriptive fields, especially in the life sciences.

This book assumes you want questions that read like real lab and lecture tasks, grade reliably, and are easy to reuse across semesters.

What this guide emphasizes

  • PGML-first authoring: write the student-facing prompt in PGML and keep regular PG as short setup code for values and answer checking.
  • Life-science style prompts: short narratives, concentrations and dilution workflows, structured data, and interpretation questions that match how biology is taught and practiced.
  • Practical reuse: stable patterns you can clone and copy-edit to build question banks quickly without breaking grading.

How to use this book (quick start)

This guide is organized as short pages you can read out of order. Pick the path that matches what you are doing today.

  • First win: start with Chapter 1.5 and copy-edit a complete PGML-first problem, then preview a few variants.
  • Setup conventions: read Chapter 2 and adopt the minimal PG scaffolding pattern for new files.
  • Prompt writing: go to Chapter 3 for PGML patterns such as blanks, lists, and tables.
  • One complete example: read Chapter 4 to see how preamble, setup, prompt, and solution fit together in a single file.
  • Interaction choice: use Chapter 5 as a catalog when you are deciding between multiple choice, matching, numerical entry, or fill-in-the-blank.
  • Biology-first patterns: use Chapter 6 for common life science templates such as dilutions, kinetics, gels, qPCR, and pedigrees.

Apply it today

  • Pick one page that matches what you are building today and copy-edit the example rather than starting from scratch.
  • Keep regular PG minimal and write student-facing sentences in PGML.
