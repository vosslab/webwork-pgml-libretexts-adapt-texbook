This page defines the short list of terms you will see repeatedly in this book. The goal is to make the later chapters easier to read without stopping to decode vocabulary.

The terms are grouped into authoring terms and platform terms. Do not memorize these; return here when a term blocks your progress.

Common terms used in this book

Term Meaning (in plain language) Why you care while authoring
WeBWorK authoring Terms you will see inside problem files. These are the knobs you touch most often.
PG The problem-file language that WeBWorK runs. In this book, PG stays minimal and supports the PGML prompt.
PGML A markup layer inside PG for writing the student prompt. Most text, structure, and blanks should live here.
Macros Reusable code libraries loaded by a problem file. Macros enable question types, answer checkers, and PGML features.
MathObjects Typed answer objects (numbers, formulas, lists) used for grading. They make checking reliable and reduce string-matching errors.
Context Rules that control what student input is allowed. Wrong Context is a common cause of unexpected grading.
Answer checker The rule used to decide whether an answer is correct. Tolerances and formats usually live here.
Seed A number that reproduces one randomized variant. Seeds let you debug the exact failing version.
ADAPT and LibreTexts platform Platform terms used in ADAPT and LibreTexts. Useful when you navigate, share, or troubleshoot.
ADAPT LibreTexts' assessment and adaptive learning system. This is where you assign questions to students and manage delivery.
WeBWorK An open-source homework system used heavily in math and science. This book focuses on WeBWorK problems that run inside ADAPT.
H5P A framework for interactive questions used in ADAPT. Useful for some prompts, but less suited to algorithmic parsing.
Commons LibreTexts' library of shared pages and resources. Where course content and shared assets are often stored.
Insight LibreTexts' knowledge base and documentation hub. Where platform help and policies usually live.
Studio LibreTexts' repository for H5P questions and media. Often where non-WeBWorK items are stored and shared.

For platform-wide definitions beyond this quick list, consult the LibreTexts glossary used by your campus.

Apply it today

  • When a term is unfamiliar, return here before you change macros or grading rules.
  • Use a fixed seed to reproduce any issue you need to debug.
