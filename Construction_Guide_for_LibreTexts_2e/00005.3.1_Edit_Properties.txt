At the top is the name of the Project which is filled in automatically from the Project Title. Next are a set of properties set by drop
down menus. The choices will be displayed on the main project page as will be shown in Section 5.4: Project Properties and Links.
The properties below are optional but it is strongly recommended that they be filled out. Project Tags and the Associated
Organizations are the most important. The Tags become the metatags for searching in Commons, the Associated Organizations are
are institutions associated with the books (e.g. your college). C-ID is a California scheme for categorizing courses.

change the order on the pages. If the Commons page is to be linked to an ADAPT homework system course insert the URL. Source
