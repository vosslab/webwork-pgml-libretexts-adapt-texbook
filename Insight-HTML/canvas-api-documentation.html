Coupling a Canvas Course to a Previously Linked ADAPT Course

Description: ADAPT now has a partial integration of the Canvas API. For instructors, this means that when instructors create assignments in ADAPT, the assignments will be automatically set up in Canvas with the c

  • Copy the course contents from your current Canvas course to a new Canvas course
  • Copy the currently linked course in ADAPT to a new ADAPT course
  • Link the new ADAPT course to the new Canvas course following the process above in “Linking a New Course”. (https://youtu.be/rBytc6ZEgls).

There are a few things to be aware of:

  • If you plan on re-using the same ADAPT course in the future, you’ll need to honor the naming convention shown in the video since ADAPT looks for its twin assignment in Canvas using the name of the assignment.  So for example, if your ADAPT assignment name is “Some Assignment”, then in Canvas, ADAPT will create “Some Assignment (ADAPT)”.  The next time you link ADAPT to a Canvas course, ADAPT will look for “Some Assignment (ADAPT)” in Canvas to link up with “Some Assignment”
  • Since you can have non-ADAPT assignments in your Canvas course, please take care in how you name them so that ADAPT doesn’t try to map an ADAPT assignment to one that really isn’t an ADAPT assignment
  • If you are a lead instructor, please don’t do the linking on behalf of your instructor cohort.  Doing so will tie your account to your instructors’ assignments even if they are not yours.
