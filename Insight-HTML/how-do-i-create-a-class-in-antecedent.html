Last updated at 03/29/2025 03:27 PM by Delmar Larsen

Description: How do I create a class in The Forge?

  1. Once you are logged in, click on the My Classes (Instructor) button.
  2. Click on the New Class button.
  3. Complete information for the class.
  4. Generate an Enrollment Key by clicking on the Autogenerate button.
  5. Distribute Enrollment Key to students, so that they can join the class once they create an account.
  6. Optional: join your own class as a student to see how the class appears to students.
