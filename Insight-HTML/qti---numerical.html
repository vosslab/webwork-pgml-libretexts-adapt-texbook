QTI Numerical is a standardized question type where learners provide a numeric answer. It allows for precise assessment by specifying exact values, acceptable ranges, or tolerances.

 

Write a question prompt which requires a numerical response, specifying the margin of error accepted in the response. Optionally provide general feedback for a correct response, an incorrect response, or any response.
