Extract the number of Sig Figs from Inexact Variable

Source: https://commons.libretexts.org/insight/extract-the-number-of-sigfigs-for-an-inexact-variable
Last updated at 10/15/2024 08:06 PM by Delmar Larsen

Description: Is there a command in your inexact macro to extract the number of sigfigs for an inexact object?

This is the "public" method.  

 

$sigFigs = $inexactValue->sigFigs();

 

For example:

$concOH = InexactValue("$concOHReal");
    $sigFigs = $concOH->sigFigs();
