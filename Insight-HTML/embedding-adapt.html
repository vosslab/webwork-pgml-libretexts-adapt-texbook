Embedding Summative ADAPT Assessments into LibreTexts Books

Source: https://commons.libretexts.org/insight/embedding-adapt
Last updated at 11/26/2024 11:31 AM by Delmar Larsen

Description: Embedding Summative ADAPT Questions into LibreTexts Textbook pages

The ADAPT system also allows the embedding of Adapt assessments in your LibreTexts book in a summative manner. This allows students to find them easier and complete the assignments without leaving the textbook. You can create a separate page for embedding assignments or add them to existing pages. See Construction Guide to learn more on page creation.

Note

Students have to be logged into their Adapt account to see the embedded assignment. Assignments won’t be visible in the print version of your book.

 

Embedding Summative Assessments

To embed a new Adapt assignment to your LibreTexts page:

  1. First add the assignment and assessments to ADAPT itself. See Adding/Importing Assignments and Adding New Assessments to learn more
  2. Copy the assignment ID. You can select the text → Right click → Copy or just click on the copy icon next to the ID

 

Figure \(\PageIndex{1}\): Copy the Adapt ID

 

  1. Open the LibreText page that you’d like to add this assessment to
  2. Click Edit (See more on page editing in the Basic Editing section of the Construction Guide)
  3. Select Adapt Icon to insert a new Adapt assessment

 

Figure \(\PageIndex{2}\): To embed Adapt assessment click on the Adapt icon 

 

  1. Type/paste your Adapt ID that you copied in Step 2 and click “OK”

 

Figure \(\PageIndex{3}\): Type or paste the Adapt ID and click OK

 

  1. You can edit the numbering if you have more than one embedded assessment. If you have 3 Adapt assessments on the same page by default they all will be labeled “ADAPT 1”, but if you change the numbering you can have “ADAPT 1”, “ADAPT 2” and “ADAPT 3”

 

Figure \(\PageIndex{4}\): You can change the numbering by editing the number in parentheses 

 

  1. Click “Save” to save your page. You should see your Adapt assessment embedded. 

 

Note: If you see an error message “You are not logged in!” you need to log in using Campus Login or (if you don’t have campus login setup) you need to open Adapt website, log into your account and then refresh the LibreText page.
