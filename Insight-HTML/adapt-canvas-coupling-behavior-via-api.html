ADAPT-Canvas Coupling Behavior via LTI-API Linking

Source: https://commons.libretexts.org/insight/adapt-canvas-coupling-behavior-via-api
Last updated at 08/23/2025 09:13 PM by Delmar Larsen

Description: This a list of behaviors associated with Canvas-LMS coupled ADAPT courses

Once an ADAPT course is linked to a Canvas course through the LTI-API interface, the following behaviors can be expected:

Setting Dates and Timing

Currently, dates and timing are managed independently in ADAPT and Canvas. There is no automatic syncing between the two systems. Instructors tend to choose from three possible modes of operation to coordinate timing between the two systems.

  • Mode 1: Canvas is the Gate Keeper
  • Mode 2: ADAPT is the Gate Keeper (recommended)
  • Mode 3: No Gate Keeper (ADAPT and Canvas perfectly synced)

See the Modes of Operation page for more details on this page: 

Adding an assignment in ADAPT course

When an instructor creates a new assignment in ADAPT, it will automatically appear in Canvas through the API connection. However, this process is one-way; changes or updates made to assignments in Canvas are not authomatically synced to ADAPT.

Delete an Assignment in ADAPT course

Deleting an assignment in ADAPT removes the corresponding assignment in Canvas.

Delete an Module in Canvas

If an instructor deletes a module in Canvas, it has no effect on the linked ADAPT course. The assignments housed in that module will still exist in ADAPT. In the future, it may be possible for ADAPT to use an API call to identify any 'phantom assignments' (i.e., assignments that still exist in ADAPT but are no longer visible in Canvas).

Changing a Name of a Module in Canvas or Assignment in ADAPT

Currently, the names of Canvas Module and ADAPT Assignments are managed independently in ADAPT and Canvas. There is no automatic syncing between the two systems. When the two systems are linked, Canvas Modules are automatically created and named after the corresponding ADAPT titles. That is currenlty the first and last time the two are connected. Changing one will not automatically change the other.

Setting Points

Currently, point values per question are managed separately in ADAPT and Canvas, with no automatic syncing between the two systems. To avoid confusion and discrepancies in grading, instructors should ensure that point settings in ADAPT and Canvas are aligned.

Will Changing the point values in Canvas that affect the points posted from ADAPT or vice versa?

No. ADAPT (like any LTI-based tool) sends only the proportion correct (a value between 0 and 1) back to Canvas. Canvas then multiplies this proportion by the total point value assigned to the corresponding Canvas assignment.

  • Changing the point value in Canvas does not affect the proportion sent by ADAPT.
  • The score students see in ADAPT remains unchanged.
  • However, Canvas will display a different numeric score if its point value is changed — because it recalculates the score using the original proportion from ADAPT multiplied by the new total points.
  • This change is not retroactive for submissions that were already passed back unless the grades are explicitly re-sent from ADAPT.

If a full re-pass of all grades is needed (e.g., after adjusting Canvas point values), it can be triggered manually — please contact the ADAPT team to initiate that process.
